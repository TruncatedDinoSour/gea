# Gea

> Simple and small text editor written in NCurses and C++

(No it's not efficient in any way)

# Building

- With clang

```bash
$ ./build.sh
```

- With GCC

```bash
$ CXX=g++ ./build.sh
```

# Installing

- Install ncurses

- Run

```bash
$ ./build.sh && sudo install -Dm755 gea.elf /usr/local/bin/gea
```
